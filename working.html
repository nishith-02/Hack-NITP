Financial Budget-1

Here you can set budget for each item or description for a month. If you have completed this step already you can skip this and can proceed to the next step.

Financial Planner-2

Here you can add the amount spent on each item or description,anytime and you can proceed to the next step after you have added all your monthly expenses.

Financial Estimate-3

Here you can interpret the total amount you have spent on an item or description and you can get the detailed information on which item or description you have saved the money and on which item you have consumed more money based on the budget you have given.
